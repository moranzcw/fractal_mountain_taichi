# Fractal Mountain

A fractal demo using taichi language.

## **Requirement**

* Python 3
* numpy
* PIL
* Taichi 0.6.7 or 0.6.8

## Usage

```bash
$ pip3 install taichi
$ git clone https://github.com/moranzcw/fractal_mountain_taichi.git
$ cd fractal_mountain_taichi
$ python3 fractal_mountain.py
```
